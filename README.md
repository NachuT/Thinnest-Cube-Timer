# Thinnest-Cube-Timer
## The worlds thinnest cube timer with capacitive touch sensing
<img width="600" height="191" alt="Screenshot 2026-06-12 at 11 31 15 PM" src="https://github.com/user-attachments/assets/94c8e33e-5e0c-48bc-b295-a776180a421e" />
\n
Here are some more details! :
- USB-C powered and for data + firmware upload
- Uses 2 capacative touch ICs conencted to large copper pads
- RP2040 with Flash and Crystal embedded
- Buttons for input and oled+leds for outputs

Please feel free to view my journal to see how I went about this project or production for the gerbers to produce it yourself.
